# Overview
The Project is built with:
- [Nuxt.js](https://nuxt.com/)
- [Typescript](https://www.typescriptlang.org/)

# Quickstart

### Install dependencies

Use pnpm to install all dependencies.

```shell
pnpm install
```

# Getting Started

First, run the development server:

```shell
cd xingxing-music
pnpm dev
```
